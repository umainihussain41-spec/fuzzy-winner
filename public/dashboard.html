🚀 Quick Setup Guide
1
Set API Keys
Copy .env.example to .env and fill in your Groq, Sarvam, and Exotel credentials.
2
Start + Expose
Run the bot server then expose it with ngrok for a public WebSocket URL. npm start ngrok http 3000
3
Configure Exotel
In App Bazaar, add a Voicebot applet to your Exophone and set the Stream URL to your ngrok wss:// address + /stream.
4
Get Free Keys
Groq (LLM): console.groq.com
Sarvam (STT+TTS): dashboard.sarvam.ai
5
Test the Bot
Call your Exophone. Asha will answer with an Indian accent and answer Exotel questions!
6
Monitor Live
This dashboard shows live transcripts, event stream, and scraper usage in real time.
